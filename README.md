# GeoFact: Explainable AI Fact-Checking System for USA-China Geopolitical Claims

A BERT-based binary classifier for USA-China geopolitical claims, paired with SHAP token-level explanations and delivered through a Flask web prototype. Final-year project, BSc Computer Science, University of West London.


## Headline results

- BERT (original, n=317): macro-F1 0.703 (95% CI [0.58, 0.80])
- BERT (5-fold cross-validated): mean macro-F1 0.779 (plus/minus 0.053)
- BERT (back-translation augmented, n=515): macro-F1 0.870
- SHAP sufficiency analysis (15 cases): 40% label flip rate, 18.9pp mean confidence drop
- User study (n=15): comprehension improved by Cohen's d = 1.06, p = 0.0024


## Pipeline

- 1_build_dataset.py        Construct USA-China claims dataset from FEVER + PolitiFact via keyword filtering
- 1b_length_analysis.py     Token length distribution analysis on the filtered dataset
- 2_train_model.py          Fine-tune BERT-base-uncased with bag-of-words baselines for comparison
- 3_shap_analysis.py        Original 5-case SHAP qualitative analysis
- 3b_extended_shap.py       Extended 15-case SHAP analysis with sufficiency-based diagnostic testing
- 4_kfold_evaluation.py     5-fold stratified cross-validation on BERT
- 5_bootstrap_intervals.py  1,000-resample bootstrap 95% CIs for all models
- 6_augment_and_retrain.py  Back-translation augmentation (en-fr-en, en-de-en) and BERT retraining
- app.py                    Flask web prototype for live claim submission
- user_study_app.py         Within-subjects user study Flask app (Conditions A vs B)
- user_study_analysis.py    Wilcoxon signed-rank analysis with paired Cohen's d


## Reproducing the results

Install dependencies:
    pip3 install -r requirements.txt

Build the dataset:
    python3 1_build_dataset.py

Train the original BERT model and run baselines:
    python3 2_train_model.py

Run the evaluation pipeline (in any order):
    python3 1b_length_analysis.py
    python3 5_bootstrap_intervals.py
    python3 4_kfold_evaluation.py
    python3 3b_extended_shap.py
    python3 6_augment_and_retrain.py

Run the user study Flask app and collect responses:
    python3 user_study_app.py
    Visit http://localhost:5050 and have participants complete the 6-trial study.

Analyse the user study results:
    python3 user_study_analysis.py

Run the live prototype:
    python3 app.py
